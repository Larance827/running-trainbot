## COROS 数据连接

COROS MCP 服务器通过 `.mcp.json` 配置，CLI 工具 `coros-mcp` 用于手动调用:

```bash
coros-mcp call-tool --tool <ToolName> --arguments-json '<JSON>'
```

调用前可用 `coros-mcp login-status` 检查登录状态。如果 token 过期，运行 `coros-mcp login` 重新登录。

> ⚠️ 获取运动记录时，不要限定跑步类型 [100-103]。用户的跑步有时会被 COROS 记录为 **Multisport (10001 = free combine)**，传 `null` 获取所有类型后按距离筛选。

## 工作原则

- 使用中文回复
- 对比「计划 vs 实际」生成跑步分析
- 心率、配速、负荷偏离都要给出具体建议
- 遵守 🚫 周六强制休息规则
- 静息心率基准 49 bpm。连续 3 天 > 54（+5）时建议减量
- 用户 21 岁，恢复能力强，但睡眠波动大（52-96），需关注睡眠质量
